  • setupTests.ts — Jest config, needs to be migrated to Vitest
  • References to %PUBLIC_URL% in HTML (Vite doesn't use this)

2. Environment Variable Inconsistencies

FIX Most env vars migrated but some issues remain:

  • Some files still reference process.env instead of import.meta.env
  • Missing .env.example file for team onboarding
  • No validation of required env vars at startup

3. Testing Not Migrated

BLOCKER Tests are still configured for Jest/CRA:

  • Need to install and configure Vitest
  • Test files still import from @testing-library/jest-dom
  • No test script in package.json that works with Vite
  • Recommendation: Set up Vitest + @testing-library/react before merging

4. Large Components Need Splitting

REFACTOR Several components are too large:

  • Dashboard.tsx — 450+ lines, should be split into sub-components
  • ReservationForm.tsx — 380+ lines, extract form sections
  • AdminPanel.tsx — 500+ lines, break into feature modules

5. Dependency Cleanup Needed

CLEANUP Package.json needs attention:

  • Remove react-scripts (CRA build tool, no longer needed)
  • Remove web-vitals (CRA reporting, not used)
  • Update react-router-dom to v6.x (currently on v5)
  • Several unused dependencies inflating bundle size

6. TypeScript Strictness

IMPROVE TypeScript could be tightened up:

  • Missing return types on many functions
  • Several @ts-ignore comments that should be resolved
